## How to use it

1. Put original image files in `images/original/`.
2. Use simple permanent English filenames, for example `product-image-01.jpg`.
3. Host this folder on a public static hosting service such as GitHub Pages, Netlify, Cloudflare Pages, Amazon S3 static hosting, or your own web server.
4. Submit the direct image URL, for example:

```text

```

## Amazon review checklist

- The image URL opens the original image file directly.
- The URL does not require registration, login, password, CAPTCHA, payment, app download, or email verification.
- The URL does not use temporary signed links or expiring tokens.
- The filename and folder path stay unchanged after submission.
- The image is publicly accessible in a private/incognito browser window.
- Use the text-free product image as the main image candidate. Feature text is better suited to secondary images.
